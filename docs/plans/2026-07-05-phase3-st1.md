# axiom Phase 3 Implementation Plan (ax::st part 1 — RNG, special fns, distributions, descriptive)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans (or subagent-driven-development) to implement task-by-task. TDD: RED build before implementation on every task.

**Goal:** Statistics substrate — own RNG (PCG64), special functions, ~15 distributions with pdf/cdf/quantile/sample, numerically stable descriptive statistics.

**Architecture:** Header `include/ax/st/*.hpp`, impl `src/st/*.cpp` (hot paths). `rng` is the single entropy source passed by reference into all `sample` calls (no global state). Special functions are free functions in `ax::st::sf`, the substrate for cdf/quantile. Quantiles: closed form where available, else Newton on cdf with bisection safeguard (internal helper — Phase 4 num not required). Distributions are small structs with parameters checked at construction (`std::invalid_argument`).

**Tech Stack:** C++23 STL only, GoogleTest, existing scripts/build.cmd.

**File map:**
- `include/ax/st/rng.hpp` + `src/st/rng.cpp` — pcg64, splitmix64, uniform, normal (ziggurat), exponential, gamma sampling primitives
- `include/ax/st/sf.hpp` + `src/st/sf.cpp` — lgamma, erf/erfc, erf_inv, incomplete gamma (P/Q), incomplete beta, log_beta
- `include/ax/st/dist.hpp` + `src/st/dist.cpp` — distribution structs
- `include/ax/st/desc.hpp` + `src/st/desc.cpp` — descriptive stats
- Tests mirror: `tests/st/rng_test.cpp`, `sf_test.cpp`, `dist_cont_test.cpp`, `dist_disc_test.cpp`, `desc_test.cpp`

**Files:**

API:

```cpp
namespace ax::st {
/** PCG64 (XSL-RR 128/64). Seeded via splitmix64 expansion of a u64 seed. */
class rng {
 public:
  using result_type = std::uint64_t;
  explicit rng(std::uint64_t seed = 0x853c49e6748fea9bULL);
  std::uint64_t next_u64();
  double next_double();               ///< uniform [0,1), 53-bit
  double uniform(double a, double b); ///< [a,b)
  std::uint64_t below(std::uint64_t n); ///< [0,n), unbiased (rejection)
  // std-compatible surface
  std::uint64_t operator()() { return next_u64(); }
  static constexpr std::uint64_t min() { return 0; }
  static constexpr std::uint64_t max() { return ~0ULL; }
};
}
```

Steps:
- [ ] Write tests: determinism (same seed → same sequence); different seeds differ; `next_double` in [0,1) over 100k draws; mean ≈ 0.5 (±0.01), var ≈ 1/12 (±0.01) on 100k; `below(6)` covers all 6 faces in 10k draws, never ≥ 6; `uniform(2,5)` bounds.
- [ ] Build → RED.
- [ ] Implement: 128-bit state via `unsigned __int128`? **No — MSVC has no __int128.** Use two u64 (hi/lo) with manual 128-bit multiply-add (same technique as bigint limbs: `_umul128` not allowed? intrinsics avoided project-wide — hand 64x64→128 via 32-bit halves, already done in `src/core/bigint.cpp`; reuse pattern, do NOT include bigint). LCG step: state = state * MUL + INC (128-bit); output XSL-RR: `rotr64(hi ^ lo, hi >> 58)`.
- [ ] Build + test → GREEN. Commit `feat(st): pcg64 rng with uniform sampling`.

```cpp
namespace ax::st {
/** One-pass Welford moments. */
struct moments {
  std::size_t n = 0;
  double mean = 0, m2 = 0, m3 = 0, m4 = 0;
  void push(double x);
  double var() const;      // sample (n-1); domain_error n<2
  double sd() const;
  double skewness() const; // g1 = m3/n / (m2/n)^1.5
  double kurtosis() const; // excess, g2
};
moments describe(std::span<const double> xs);
double quantile(std::vector<double> xs, double p);  // R type-7, copies+nth_element
std::vector<double> quantiles(std::vector<double> xs, std::span<const double> ps);
mat covariance(const mat& x);   // columns = variables, rows = observations
mat correlation(const mat& x);
}
```
