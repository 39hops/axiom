# axiom Phase 4 Implementation Plan (ax::num — quadrature, ODE, roots, optimization)

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:executing-plans. TDD: RED before implementation on every task.

**Goal:** Numeric methods module: root finding, adaptive quadrature (Gauss-Kronrod + tanh-sinh), adaptive RK45 ODE integration, 1-d and multi-d optimization.

**Architecture:** Free functions taking `std::function`-free callables (templates in headers where trivial, `std::function<double(double)>` in impl where iteration-heavy — clarity over micro-perf; hot loops stay tight). Results returned as small structs with convergence flags; failure to converge throws `std::runtime_error` only where no sensible partial result exists. Multi-d optimization uses `ax::la::vec`/`mat`.

**Tech Stack:** C++23 STL, GoogleTest, ax::la for vec/mat.

**File map:**
- `include/ax/num/root.hpp` + `src/num/root.cpp` — bisection, brent, newton
- `include/ax/num/quad.hpp` + `src/num/quad.cpp` — adaptive G7-K15, tanh-sinh
- `include/ax/num/ode.hpp` + `src/num/ode.cpp` — RK45 Dormand-Prince adaptive
- `include/ax/num/opt.hpp` + `src/num/opt.cpp` — brent 1-d min, nelder-mead, bfgs
- Tests mirror: `tests/num/root_test.cpp`, `quad_test.cpp`, `ode_test.cpp`, `opt_test.cpp`

### Task 1: root finding

API (`namespace ax::num`):

```cpp
struct root_result { double x; double fx; int iters; bool converged; };
/** Bisection on [a,b], f(a)f(b) < 0 required (std::invalid_argument). */
root_result bisect(const std::function<double(double)>& f, double a, double b,
                   double xtol = 1e-12, int max_iter = 200);
/** Brent (inverse quadratic + secant + bisection), same contract. */
root_result brent(const std::function<double(double)>& f, double a, double b,
                  double xtol = 1e-14, int max_iter = 200);
/** Newton with user derivative; falls back on runtime_error if diverges. */
root_result newton(const std::function<double(double)>& f,
                   const std::function<double(double)>& df, double x0,
                   double xtol = 1e-14, int max_iter = 100);
```

Tests: cos x = x root ≈ 0.7390851332151607 (all three); x³-2x-5 Brent root ≈ 2.0945514815423265 (Wikipedia Brent example, verify vs f(root)≈0); bracket violation throws; newton on x²-2 → √2; bisect converged flag false when max_iter=3.
Commit `feat(num): root finding (bisection, brent, newton)`.

### Task 2: quadrature

```cpp
struct quad_result { double value; double error_est; int evals; };
/** Adaptive Gauss-Kronrod G7-K15 on [a,b]. */
quad_result integrate(const std::function<double(double)>& f, double a,
                      double b, double abstol = 1e-10, double reltol = 1e-10,
                      int max_depth = 50);
/** tanh-sinh for endpoint singularities on (a,b). */
quad_result integrate_ts(const std::function<double(double)>& f, double a,
                         double b, double tol = 1e-10, int max_level = 12);
```

G7-K15 nodes/weights hardcoded (standard 15-point Kronrod table, cite values from QUADPACK). Recursive bisection on error > tol.
Tests (exact closed forms): ∫₀¹x²=1/3 (1e-13); ∫₀^π sin=2 (1e-12); ∫₋₁₀¹⁰ e^{-x²}=√π (1e-10); oscillatory ∫₀^{20} sin(10x) = (1-cos200)/10; tanh-sinh: ∫₀¹ 1/√x = 2 (1e-9), ∫₀¹ ln x = -1 (1e-9); a>b sign flip.
Commit `feat(num): adaptive Gauss-Kronrod and tanh-sinh quadrature`.

### Task 3: ODE — RK45 Dormand-Prince

```cpp
struct ode_result {
  std::vector<double> t;                 ///< accepted step times
  std::vector<la::vec> y;                ///< states at t
  int steps = 0, rejected = 0;
  bool converged = true;
};
/** Integrate y' = f(t,y) from t0..t1, adaptive Dormand-Prince RK45. */
ode_result solve_ivp(
    const std::function<la::vec(double, const la::vec&)>& f, double t0,
    double t1, la::vec y0, double reltol = 1e-8, double abstol = 1e-10,
    double h0 = 0.0 /* auto */);
```

Butcher tableau: standard DP(4,5) coefficients. Error norm: RMS of component errors scaled by abstol + reltol*|y|. Step accept if err<=1, new h = h*clamp(0.9*err^{-1/5}, 0.2, 5).
Tests: y'=y, y(0)=1 → y(2)=e² (rel 1e-7); harmonic y''=-y as 2-d system over 10π: back to start (1e-6), energy drift < 1e-6; y'=-2ty, y(0)=1 → e^{-t²} at t=3; stiff-ish λ=-50 decay still correct (more steps, converged).
Commit `feat(num): adaptive RK45 (Dormand-Prince) ODE solver`.

### Task 4: optimization

```cpp
struct min_result_1d { double x, fx; int iters; bool converged; };
/** Brent minimization (golden + parabolic) on [a,b]. */
min_result_1d minimize(const std::function<double(double)>& f, double a,
                       double b, double xtol = 1e-10, int max_iter = 200);

struct min_result {
  la::vec x; double fx; int iters; bool converged;
};
/** Nelder-Mead simplex. */
min_result nelder_mead(const std::function<double(const la::vec&)>& f,
                       la::vec x0, double ftol = 1e-10, int max_iter = 2000);
/** BFGS with numeric central-difference gradient (or user gradient). */
min_result bfgs(const std::function<double(const la::vec&)>& f, la::vec x0,
                double gtol = 1e-8, int max_iter = 500);
min_result bfgs(const std::function<double(const la::vec&)>& f,
                const std::function<la::vec(const la::vec&)>& grad,
                la::vec x0, double gtol = 1e-8, int max_iter = 500);
```

BFGS: inverse-Hessian update, Armijo backtracking line search. Numeric gradient: central diff h = cbrt(eps)*max(1,|x_i|).
Tests: 1-d (x-2)²+3 → x=2 (1e-8); x⁴-3x³+2 interior min; Rosenbrock (a=1,b=100) from (-1.2,1) → (1,1): NM ftol loose (1e-6 coords), BFGS tighter (1e-5); 5-d quadratic bowl xᵀAx with SPD A → 0 (BFGS, 1e-6); sphere from far start NM; non-converged flag when max_iter tiny.
Commit `feat(num): optimization (brent 1-d, nelder-mead, bfgs)`.
